Skills

Frontend

I specialize in building clean and interactive user interfaces with React and Tailwind CSS, ensuring both performance and design consistency. I also explore React Three Fiber for 3D experiences and enjoy blending creativity with technology to make visually engaging web applications.

Backend

On the backend, I work primarily with Node.js and Express to build robust, scalable APIs. I focus on writing clean, maintainable code and creating systems that can handle real-world workloads. For AI pipelines, I also work with Flask, integrating machine learning models directly into applications.

Databases

I’m experienced with MongoDB as my go-to database for flexible, document-based storage. Alongside it, I use Mongoose for schema design and queries. I’m also comfortable working with relational databases like PostgreSQL when projects call for structured data.

Languages

My main programming languages are JavaScript and Python. JavaScript powers my web development work, while Python is central to my AI and machine learning projects. I also have working knowledge of C++ from my academic foundations, which helps with problem-solving and efficiency.

AI & Machine Learning

My background in AI includes working with Whisper for speech recognition, IndicTrans for translation, Coqui TTS for text-to-speech, and PyAnnote for speaker diarization. I also actively study transformer architectures and large language models to deepen my understanding of how modern AI systems are built.

Tools & Workflow
